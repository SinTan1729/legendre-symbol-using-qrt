# legendre-symbol-with-qrt
This is part of a Directed Reading Program
The goal of this code is to make a function using the quadratic reciprocity theorem (QRT) that provides the quadratic residues of a givin Legendre symbole. 
The QRT states that for a given Legendre symbole (q/p) where q and p are distinct primes: (q/p)=(p/q) except if q or p is congruent to 3 mod(4) then (q/p)=-(p/q). In other words, if q and p are distenct primes then we can flip the Legendre symbole, with the exeption of 3 mod(4) where we can still flip it but need a negative sign in front.
